Methods

animate(images, timeInterval, conditionFunction, playAnimationOnce)

Adds an array to the loopArray of the global World object to generate an animation for the object on which this method is called, using the image paths provided as an argument. Optionally a condition for playing the animation can be provided. Make sure to load the images in the image Cache of the object before using this function!

Parameters:
Name Type Default Description
images Array

Array of URL's for images given as strings
timeInterval number

Time interval for playing the animation
conditionFunction function

Function containing a condition for starting the animation and returning a boolian value, default value is a function always returning true
playAnimationOnce boolean false

When true the animation will only be played once, defealt value is false.
